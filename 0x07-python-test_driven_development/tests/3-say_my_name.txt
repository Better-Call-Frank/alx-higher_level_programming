# 3-say_my_name.txt

===========================
How to Use 3-say_my_name.py
===========================

This module includes a function called "say_my_name(first_name, last_name="")" for displaying a person's name.

Usage
=====

The "say_my_name(...)" function prints "My name is <first_name> <last_name>".

Example:
```python
>>> say_my_name = __import__('3-say_my_name').say_my_name
>>> say_my_name("Brennan", "Baraban")
My name is Brennan Baraban
```

Example:
```python
>>> say_my_name("Cornelius Maxwell", "Jones II")
My name is Cornelius Maxwell Jones II
```

The "last_name" parameter is optional. If no last name is provided, an empty string is printed instead.

Example:
```python
>>> say_my_name("Brennan")
My name is Brennan
```

Invalid Names
=============

Both the "first_name" and "last_name" parameters must be strings. Otherwise, a TypeError is raised.

Examples:
```python
>>> say_my_name(6, "James")
TypeError: first_name must be a string

>>> say_my_name("LeBron", ["Cavs", "Lakers", "Heat"])
TypeError: last_name must be a string

>>> say_my_name({"LeBron": 6, "James": 23}, 7.7)
TypeError: first_name must be a string

>>> say_my_name(None)
TypeError: first_name must be a string
```

At least one name must be provided.

Example:
```python
>>> say_my_name()
TypeError: say_my_name() missing 1 required positional argument: 'first_name'
```
