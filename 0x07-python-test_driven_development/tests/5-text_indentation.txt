# 5-text_indentation.txt

================================
How to Use 5-text_indentation.py
================================

This module contains a function called "text_indentation(text)" for formatting text with indentation.

Usage
=====

The "text_indentation(...)" function prints text with two new lines after each occurrence of the characters ".", "?", and ":".

Example:
```python
>>> text_indentation = __import__('5-text_indentation').text_indentation
>>> text_indentation("Hello?")
Hello?
<BLANKLINE>
```

No spaces are printed at the beginning of a line.

Example:
```python
>>> text_indentation("   Hi there.")
Hi there.
<BLANKLINE>
```

Example:
```python
>>> text_indentation("          ")
```

Similarly, no spaces are printed at the end of each printed line.

Example:
```python
>>> text_indentation("Hello.   ")
Hello.
<BLANKLINE>
```

Example:
```python
>>> text_indentation("    Woah now.    This is messy.   ")
Woah now.
<BLANKLINE>
This is messy.
<BLANKLINE>
```

New lines are only printed for the characters ".", "?", and ":". Text that doesn't end with one of these characters is not followed by a new line.

Example:
```python
>>> text_indentation("No ending period, what bad grammar")
No ending period, what bad grammar
```

New lines within a string are printed as usual.

Example:
```python
>>> text_indentation("Let's print a new-line! Here goes:\nPrinted.")
Let's print a new-line! Here goes:
<BLANKLINE>
<BLANKLINE>
Printed.
<BLANKLINE>
```

Example:
```python
>>> text_indentation("\n\n\n We just printed three new lines.")
<BLANKLINE>
<BLANKLINE>
<BLANKLINE>
We just printed three new lines.
<BLANKLINE>
```

Example:
```python
>>> text_indentation("A sneaky \n new line.")
A sneaky 
new line.
<BLANKLINE>
```

Invalid Text
============

The "text" parameter must be a string. Otherwise, a TypeError is raised.

Example:
```python
>>> text_indentation(7)
TypeError: text must be a string
```

Example:
```python
>>> text_indentation({"one": 1, "two": 2})
TypeError: text must be a string
```

Example:
```python
>>> text_indentation(None)
TypeError: text must be a string
```
